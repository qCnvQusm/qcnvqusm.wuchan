信的抬头是“兄长”，落款是“啸卿”。龙文章眉梢挑起，在脑海中仔细搜索一番，确信虞啸卿是家里的长子，并无兄长。于是接着往下看。信上第一句是：“你走了已有七日，我夜夜梦到你。”龙文章静了一息，猛地反应过来，原来这“兄长”叫的是自己。这一发现仿佛将龙文章惊了一跳，他猛地将信纸反拍在桌上，瞪着信纸背面隐约透出的字迹，狠狠地吸进一口气，又颤抖着吐出，如此几次，才俯下身略显笨拙地将信纸原样塞回文书堆里。 龙文章长舒一口气，跌回椅子里，摊手摊脚地坐着，只用眼睛到处乱瞄，接着他顺理成章地看到桌下那只带锁的抽屉。这种简陋的锁根本难不倒龙文章，他随手摸来一截铁丝，在锁孔里捅了几下就如愿听到锁舌弹开的清脆“咔嗒”声。龙文章满意地翘起嘴角，探着身子拉开抽屉，发现里面放着的不过是几把枪。 龙文章拿起放在最外面的南部十四。王八盒子握在手里的触感有些熟悉。于是他将手枪举到眼前，果然看到枪柄上写着“立花奇雄”。在缅甸时这把枪差点要了他的命，第一次在禅达见到虞啸卿时他将这把枪送给虞啸卿，最终也是这把破枪将那颗幸运弹送进了自己的脑袋。就像他曾经将自己的命毫无保留地交给虞啸卿，他欠虞啸卿的命也已在南天门上还尽。所以他将自己的命拿回来，然后自由地离开。 柯林斯曾万分不能理解龙文章的配枪怎么会如此肮脏，如果此时他看到这把南部十四一定不会相信这曾是龙文章的配枪——尽管后来大部分时间它确实在虞啸卿手中。但这把枪实在干净得有点过了头，仿佛有人日日都在仔细擦拭，即使它那该死的性能根本配不上如此细致的保养。 枪被拿起，于是原本折起来压在枪下的纸片得以重见天日。它迫不及待地舒展，果然成功地引起了龙文章的注意。 龙文章顺手把枪别在腰里，用两根手指捏起那张被折叠的纸片，拿到眼前展开来。这张已经皱皱巴巴的纸原来是一封军里发来的电报，上面却只有四个字：“攻击立止。”纸上还有几处水痕，像是谁落下的泪水。龙文章异常平静地盯着报文看了一会儿，照着折痕将电报原样折好放回抽屉里，又抖着手从腰里摸枪，压在纸上时却发现那是一把沾着泥土的柯尔特。南部十四被他别在了另一边。 龙文章又拾起那把的柯尔特，拽着袖口仔仔细细地擦拭一遍。这把脏兮兮的枪才像是被龙文章佩过的样子，倒不像是虞啸卿的配枪。或者说，这把枪到了龙文章手里之后才变得万分埋汰。龙文章将腰里的南部十四抽出来压在折起的电报上，又将柯尔特端正地摆在旁边，盯了半晌才将抽屉小心翼翼地关上。 龙文章很快发现了窗台上的半瓶洋酒和一个牛肉罐头。龙文章品不来美国人的酒，但牛肉罐头还是不错的，于是自顾自开了罐头缩在虞啸卿平日里不怎么坐的椅子上吃了。 窗外天色渐晚，龙文章已经将虞啸卿的房间翻了个遍，于是又将自己摊回椅子里熬时间，却忍不住一点一点回忆过去。 事如春梦了无痕 龙文章死后，在虞啸卿每日短暂的睡眠时间里，几乎时时都有龙文章的身影。梦里的龙文章有时腆着脸涎笑着讨要物资，在他面前大叫“漫天要价就地还钱”。有时在南天门上苦守，而他与龙文章并肩仰倒在地上，一同战死在南天门。更多的时候是像现实一样，龙文章在他面前饮弹自尽，然后那张仿佛总也洗不干净的脸开始燃烧，生命迅速流失时龙文章仍在呢喃。虞啸卿抱着龙文章的身体努力将耳朵贴近他缓慢张合的嘴巴，却什么也听不分明。 这个梦太真实，以致虞啸卿醒来时甚至仍然觉得自己被熊熊火焰灼烧，手上似乎还残留着鲜血的粘稠触感。不过这也只是个梦罢了。虞啸卿醒来之后也只是在床上躺着缓了片刻便照常起身。 对虞啸卿来说，这一天与过去的每一天相比并没有什么不同。禅达的清晨像过去的每一天一样如期而至，虞啸卿也像过去的每一天一样开车到山顶。虞啸卿在平日练刀的地方停下车时，路边叶子上的露水还在缓慢地滚动，天色仍沉沉地发暗。不过这时恐怕是虞啸卿每天最安逸、最放松的时刻。他静静地坐在车上，听山林间的鸟鸣，感受清晨拂过面颊的微风，呼吸与山下截然不同的清澈空气。 虞啸卿并没有坐很久。他今天不是来呆坐着享受安逸的。可虞啸卿也没有像往常一样带着那把刀，反而拿着一小瓶美国人带来的洋酒下了车。所以今天与过去还是有一些不同——今天是龙文章的头七。 虞啸卿大部分时间不在乎人是不是真的有其魂，小部分时候希望人真的有魂。此刻就是虞啸卿的小部分时候。虞啸卿希望龙文章的魂魄能在随便什么的指引下回家，尽管他并不确定禅达对龙文章来说是否算得上一个家。 山顶上常年无人打扰，生着一棵高大的铁杉，树下开着一丛艳丽的石斛兰。虞啸卿在将酒放在花丛边，然后直挺挺地跪下去，垂头静了片刻，探着身子打开酒瓶，从腰间拿出一把南部十四，又从胸口的口袋里拿出一块洁白的手帕，将那把锃亮的枪从里到外仔细擦拭一遍，小心地摆在地上。 虞啸卿白日里向来不允许自己沾酒，此时也只是将瓶口举到鼻尖闻了闻，抬手将大半瓶酒浇在地上。 “兄长。也许你并不愿意我叫你‘兄长’，但我真的把你当兄长看……”虞啸卿忍不住笑了一下，“事已至此，让我过过嘴瘾吧。” 虞啸卿垂着头，眼眶泛红，轻声道：“兄长，我知道，我欠着你的。 “今日是你的头七。你说你是招魂的，你的魂魄今日可会回来？我该去祭旗坡看你的，但我不能、我没法去。”虞啸卿的眼泪砸进土地里，与刚刚倒下去的酒混在一起，“你看，我总是这样，什么都做不到，大概你会觉得我天真得可笑。” 平日再无人可诉的话，此刻虞啸卿对着一把破枪大讲特讲，讲自己的童年到青年，讲龙文章与虞啸卿，讲过去说不出口的依恋。 天光即将大亮，虞啸卿的泪水已经止住，他并不打算多留，重新戴上手套蹭蹭眼睛，捡起那把南部十四摩挲一遍别回枪套里，没什么留恋地起身离开。 这几日缠着他的翻来覆去总是那些事，西岸的部署、物资、火力，也还有人揪着授勋那日不放，非要说他通共。虞啸卿看了半日公文，总觉得不痛快，忍不住抽了张信纸来给龙文章写信。写完一封短信，走到窗前去，将清晨剩下的半瓶酒摆在窗台上，翻翻找找又拿了一个牛肉罐头放在酒瓶边上，喃喃自语：“回家吧，兄长。” 军部的作战会议并不会因为虞啸卿的厌倦就取消，虞啸卿和友军开完会回到指挥部时已经入夜，他头昏脑涨地回屋关门，全未察觉到房间里还有另一个人存在，背后毫无防备地遭到袭击。 龙文章一条手臂将虞啸卿紧紧箍在自己怀里，一手捂住虞啸卿的嘴，凑在虞啸卿耳边压着声音：“师座、师座，是我，龙文章，我是龙文章！” 虞啸卿如遭雷劈，马上就要送出的肘击戛然而止，紧绷着身体定在原地。龙文章这才慢慢拿开手，脸上挂起涎笑转到虞啸卿面前：“师座安好，师座无恙，我一直惦记着师座。” 虞啸卿在黑暗中抿唇瞪眼直盯着他：“你没死。你从哪里来？” 龙文章状似忸怩地低下头：“师座还当我是妖孽，吞了枪也不会死。” 虞啸卿伸手要摸他的脸，龙文章却双手捂脸矮下身子后退半步躲开：“可我不是妖孽。我死了，师座……军座，”龙文章像是刚想起来似的改口，“我死了。但是醒来就在军座房里。” 虞啸卿悻悻地收回手：“还当你能听到我说话。” 于是龙文章又凑上来：“师座叫我了？那我肯定是听到师座叫我才回来的。” 虞啸卿哼笑一声，上前一步强硬地将龙文章拢在怀里：“鼓唇弄舌。” 龙文章闭上嘴巴，沉默地被虞啸卿拥着，感觉到气氛变得微妙，不安地动了动。虞啸卿难得从善如流地放开他，红着眼睛借着从窗口漏进来的月光上下打量龙文章——像过去常常看到的那样，军装衬衫上套着从自己身上扒走的麦基诺大衣，袖子卷起，露出一截精壮的手臂，左手手腕上仍然戴着那只捡来的手表。毛茸茸的头发被月光染上一层白霜，仿佛很不好意思似的地驼背低头从下往上觑人。 这样的熟悉感让虞啸卿十分放松，他回身摁亮了屋顶的电灯。 “你到底怎么回事？”虞啸卿踱到书桌边看着随他转身的龙文章。 “我也不知道，一睁眼就在这里了……也许就是师座叫我回来的，头七回魂师座总知道吧？”龙文章挨到虞啸卿身边，“我这上颚还留着弹孔呢。” 虞啸卿扭头，仿佛不可置信似的瞪着他，眼珠缓慢地动了两下，眼神黯淡下去：“抱歉。我没办法了。” 龙文章不安地扭了一下身子：“别说这，师座，都过去了，都……来不及了。” 虞啸卿有些颓唐地坐到椅子上：“你该恨我的。可是我从来没想过要给你一发子弹。” 龙文章看着他，少有地露出悲戚：“我明白师座的难处。可对就是对错就是错，师座，对错很重要。还记得我那个草包副官说的吗，卧薪尝胆也是一种鸦片。别骗自己。” 虞啸卿瞥了他一眼，看着窗外出神：“如今东西两岸的军队全听我部署，我们会一路西进。” “这就是师座该做的，也是师座想做的。” 虞啸卿却笑了：“有时候我真想不明白。我以为手里握着更多东西就能做我想做的，现在才发现原来不是这样。” “世事如此，师座。没有一件事会等我们做好万全的准备再去做，也没有人能真的做好万全的准备。” 虞啸卿红着眼睛猛地转头看他：“你走了才七天，我恐怕后悔了八百回，我只知道你和我一样想做事，却没想过你不是我，逼得你为了断我念想断自己后路做到这一步。如果我能安静下来听你说话，可能结果就不一样。你被关着时我不敢见你，因为我没法给你一个像样的回答，现在我后悔得要命，如果我能不要脸地去见你，可能结果又不一样。” 龙文章静了一静才说：“……我原本倒想活上一万年呢，瞧尽人间。从东到西从南到北，做完事了就去看看咱们事做得怎么样。不过我是看不到了，师座，你还要做你该做的事，还要去瞧瞧这大好河山。” “抱歉。”虞啸卿垂下眼睛。 龙文章倚着桌沿没看虞啸卿：“上南天门之前我骗自己，拿下南天门就能还清过去欠下的一千座坟，上去之后我就没法再骗自己，我不但还不上，还欠下了更多。” 虞啸卿眼睛红得几乎要落下泪了，他说：“抱歉。南天门上的债是我欠的，几千座坟合该由我来背。我会还债的。” “我早就知道师座和别人不一样。可是几千座坟很重，师座背得住吗？”龙文章低头看他，“如果还有机会还有时间，我肯定愿意和师座一起做我们该做的事。可是没机会了。没机会了……剩下的事得师座自己想、自己做。” 虞啸卿瞪着他，很不服气似的：“怎么没机会？怎么没机会！你给我好好看着，看我怎么做事。” 龙文章挂起涎笑：“是没机会了，师座。我已经死了，师座不会忘了吧？”又很兴奋似的把脸凑过去，“师座现在把我剖开应该还能找到一颗子弹呢。师座要不要看看弹孔？师座还没见过吧？” 虞啸卿皱眉狠狠地瞪着他，咬牙咬得腮帮子鼓起，强忍住了扇人一巴掌的冲动，伸手卡着龙文章的下颌将那张狗脸拽得更近。 “张嘴。” 龙文章乖乖张开嘴巴。虞啸卿就把手指头伸进去顺着上颚摩挲。虞啸卿摸得很慢，龙文章忍不住顶起舌头吞口水。在包裹着手指的布料变得潮湿时虞啸卿摸到了那枚弹孔。龙文章含着虞啸卿的手指忍不住哼哼，想把脑袋挪走。虞啸卿却更用力地扣着他的下巴，两指捉住那条湿漉漉的舌头摩挲。 龙文章眼里泛着水光，顺从地将身体也蹭得更近，一手扶住桌沿，用下体下流地顶了顶虞啸卿的侧腰，牙齿轻轻咬住他的手指，舌尖隔着手套在指腹流连。 虞啸卿的呼吸不由得变得急促，猛地抽出手指，将龙文章的脸推开。龙文章从善如流地蹲在虞啸卿腿边，从下向上觑着他的脸：“师座也想要了，别骗自己。” 虞啸卿太阳穴直跳，看着龙文章蹭得更近，嘴唇贴上自己的裆部。他盯着龙文章的眼睛，不可抑制地勃起，马裤紧绷的束缚感让他焦躁。 龙文章看到虞啸卿泛红的眼睛里情欲混着难堪，忍不住调笑：“师座这条马裤可不好脱。” 虞啸卿瞪他一眼，扯着后领将人拽到床边，背对着龙文章将身上的披挂逐一卸下，很快脱得只剩一条底裤，愤愤转过身来坐在床上：“脱。” 龙文章看着虞啸卿凝重得像要去开作战会议一般的神情，忍不住笑出声：“师座还是这样。师座应该放松点。”被虞啸卿一瞪立马敛了笑，飞快地蹬下自己裤子，边解衣扣边挨过去含住虞啸卿的唇珠。
